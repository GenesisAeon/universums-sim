# Programmstruktur

Dieses Dokument beschreibt die Grundstruktur von Nullmembran, Feldern, Teilchen und der Evolution innerhalb der Simulation.

- **Nullmembran**: Fundamentales Bezugssystem.
- **Felder**: Verteilen Kräfte und Informationen.
- **Teilchen**: Wechselwirken innerhalb der Felder.
- **Evolution**: Regelt die zeitliche Entwicklung dieser Elemente.
