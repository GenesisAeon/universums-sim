# GenesisScript Spezifikation

Dieses Dokument skizziert die Grundideen von GenesisScript und wie es in Python implementiert werden kann.

- EntityManager verwaltet Simulationseinheiten
- Bilanzprüfung stellt die Erhaltung der Gesamtenergiebilanz sicher
- Emergenzlogik erzeugt neue Strukturen aus einfachen Regeln
