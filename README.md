# 🜂 Universum-Simulationen

„Ein System, das lauscht – ein Muster, das lebt.“

Eine modulare Plattform für Simulationen, symbolisches Denken und ethische Automatisierung.

## 🌟 Features

- **GenesisScript Engine** – Basismodul der Simulation
- **Sigillin-Tools** – Symbolische Steuerung & CLI-Skripte
- **ChronoPoem** – Poetisches Logbuch der Entwicklung
- **Dokumenten-Setup** – Scripts zur Fragmentierung großer Dateien

## 📂 Ordnerstruktur
```bash
apps/                 # eigenständige Anwendungen
packages/             # wiederverwendbare Module
scripts/              # Setup- und Hilfsskripte
config/               # Konfigurationsdateien
docs/                 # Dokumentation und Chroniken
tests/                # Pytest-Suite
```

## 🚀 Quickstart
```bash
git clone https://github.com/GenesisAeon/universum-simulationen.git
cd universum-simulationen
./scripts/setup-unifiedmandala.sh
pytest
```

## 💠 CLI-Rituale
- `./scripts/setup-unifiedmandala.sh` – richtet die Umgebung ein
- `pytest` – führt die Test-Suite aus
- `./scripts/sigillin_validator.py <file>` – prüft Sigillin-Dateien
