# Universum-Simulationen

Dieses Repository sammelt Konzepte und Skripte für das Projekt "Universum-Simulationen". Ziel ist es, eine modulare Umgebung für unterschiedliche Simulationen aufzubauen.

## Struktur

- **apps/** – Platz für eigenständige Anwendungen
- **packages/** – Wiederverwendbare Bibliotheken
- **scripts/** – Hilfsskripte wie das Setup
- **config/** – Konfigurationsdateien
- **docs/** – Dokumentation und Chroniken

Weitere Informationen findest du in `Handbuch.md` sowie in der `GenesisChronik.md`.
